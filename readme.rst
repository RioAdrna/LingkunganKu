LingkunganKu adalah aplikasi web berbasis CodeIgniter 3 yang digunakan untuk mempermudah masyarakat dalam menyampaikan laporan atau pengaduan terkait masalah lingkungan.
Website ini menyediakan alur pelaporan yang jelas mulai dari masyarakat, diproses oleh petugas, hingga dikelola oleh admin.

Tujuan utama dari aplikasi ini adalah menciptakan platform pengaduan yang cepat, mudah, dan transparan agar masalah lingkungan dapat ditangani dengan lebih efektif.
